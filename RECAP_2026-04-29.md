# PerfumeRAG Build Recap

Prepared on `2026-04-29`.

## What was built

### Official site scraper

Implemented a reusable scraper framework under `scripts/official_scraper/` with adapters for:

- `Guerlain`
- `Xerjoff`
- `Zara`

The runner writes:

- raw snapshots under `data/raw/<brand>/<run-id>/`
- normalized product records under `data/official-products/*.jsonl`
- latest release summary at `data/official-products/latest-release-enrichment.csv`

## What works

### Guerlain

- official list discovery via Demandware grid endpoint
- direct product page scraping
- JSON-LD parsing with metadata fallback
- raw HTML snapshots

### Xerjoff

- collection discovery from official new-arrivals page
- product extraction via Shopify `.js` endpoints
- structured sizes, price, description, and tag capture

### Zara

- official latest-release seed capture
- blocked-state detection and preservation

## Main blocker

`Zara` product pages return `403` in this environment, so the scraper stores those as blocked records and preserves curated seed metadata instead of failing the pipeline.

## Useful commands

```bash
npm run brand-registry
npm run official-scrape
npm run build-rag
python3 scripts/query-rag.py 'woody winter vanilla' --limit 5
python3 scripts/query-rag.py 'xerjoff new arrivals tropical fruits' --limit 5
```
